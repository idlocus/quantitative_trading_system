## 1. 核心理念

### 1.1 技术分析三大假设
1. **市场行为反映一切** - 价格消化所有已知信息
2. **价格沿趋势移动** - 趋势在反转前保持不变
3. **历史会重演** - 价格形态和信号会重复

### 1.2 第四原则：纪律优先于信念
- 策略规则优先于当前判断
- 定期评估策略有效性
- 限制持仓时间，避免"亏损死扛"

### 1.3 分析前先问
- 市场处于趋势还是震荡？
- 是领先还是滞后？
- 是否涨/跌太快太远？

## 2. 多时间框架分析 (MTFA)

### 2.1 时间框架层级
| 时间框架 | 用途 | 说明 |
|---------|------|------|
| 月线 | 定义主要趋势 | 长期方向锚点 |
| 周线 | 细化主要趋势 | 识别中期形态 |
| 日线 | 战术执行 | 精确定位入场/出场 |

### 2.2 趋势判断规则
- **上升趋势**: 均线多头排列 (短期 > 中期 > 长期)
- **下降趋势**: 均线空头排列 (短期 < 中期 < 长期)
- **震荡**: 价格在均线间来回穿梭

### 2.3 时间框架一致性
- 多时间框架方向一致时 → 信心增加
- 多时间框架冲突时 → 降低仓位或观望

## 3. 技术指标分类与应用

### 3.1 趋势指标 (用于趋势市场)

| 指标 | 周期参数 | 信号 |
|------|---------|------|
| SMA/EMA | 10/50/200 | MA金叉/死叉 |
| MACD | 12/26/9 | MACD与Signal交叉 |
| DMI+ADX | 14 | ADX>25确认趋势 |
| Ichimoku | 9/26/52 | 价格在云上/下 |

**买入信号**:
- 短期MA上穿中期MA (金叉)
- MACD线在Signal线上方
- ADX上升 > 25

**卖出信号**:
- 短期MA下穿中期MA (死叉)
- MACD线在Signal线下方
- ADX下降

### 3.2 动量指标 (用于震荡市场)

| 指标 | 周期参数 | 超买/超卖 |
|------|---------|----------|
| RSI | 14 | 70/30 |
| Stochastic | 14/3 | 80/20 |
| ROC | 12 | -10/+10 |
| CCI | 20 | 100/-100 |

**RSI背离交易**:
- 底背离 (价格新低, RSI更高) → 买入
- 顶背离 (价格新高, RSI更低) → 卖出

### 3.3 波动率指标

| 指标 | 用途 | 信号 |
|------|------|------|
| Bollinger Bands | 识别突破 | 带宽收缩后扩张 |
| ATR | 设置止损 | 动态跟踪波动 |
| Keltner Channels | 趋势确认 | 价格突破通道 |

**布林带应用**:
- %B > 0.9: 价格触及上轨，可能回调
- %B < 0.1: 价格触及下轨，可能反弹
- Bandwidth < 2: 波动率低，预示突破

### 3.4 成交量指标

| 指标 | 公式 | 信号 |
|------|------|------|
| OBV | 累计(涨:+V, 跌:-V) | OBV与价格背离 |
| CMF | 累计MFV/累计Vol | CMF穿越0 |
| MFI | 资金流量比率 | 80/20 |
| VWAP | Price×Vol加权 | 价格在VWAP上/下 |

### 3.5 市场广度指标

| 指标 | 用途 |
|------|------|
| A-D Line | 衡量市场参与度 |
| McClellan Osc | 识别趋势变化 |
| Arms Index | 资金轮动 |

## 4. 支撑与阻力

### 4.1 支撑阻力识别
1. **历史高低点** - 重复测试的水平
2. **均线** - 动态支撑/阻力
3. **布林带** - 轨道支撑
4. **斐波那契回撤** - 23.6%, 38.2%, 50%, 61.8%

### 4.2 角色互换
- 阻力突破 → 变成支撑
- 支撑跌破 → 变成阻力

### 4.3 趋势线
- **有效趋势线**: 多次测试、30-50度斜率
- **趋势线突破确认**: 收盘价突破1%以上

## 6. 综合评分系统

### 6.1 评分权重
```
综合评分 = 趋势得分 × 40% + 动量得分 × 35% + 波动率得分 × 25%
```

### 6.2 信号判定
| 综合评分 | 建议 |
|---------|------|
| ≥75 | 强烈买入 |
| 60-74 | 买入 |
| 45-59 | 持有 |
| 30-44 | 卖出 |
| <30 | 强烈卖出 |

### 6.3 趋势判断
| 评分 | 方向 |
|------|------|
| ≥70 | 强势上升 |
| 55-69 | 上升 |
| 45-54 | 中性 |
| 30-44 | 下降 |
| <30 | 强势下降 |

## 7. 风险管理

### 7.1 止损设置
- **固定止损**: 入场价 ± 2×ATR
- **支撑止损**: 关键支撑位下方
- **移动止损**: 跟随价格创新高/低

### 7.2 仓位计算
```
仓位 = (账户2%风险) / (入场价 - 止损价)
```

### 7.3 止盈策略
- **固定目标**: 风险收益比 2:1 或 3:1
- **移动止损**: 保住部分利润
- **分批出场**: 部分持仓跟踪趋势

### 7.4 风险收益比评估
| 比率 | 说明 |
|------|------|
| ≥3:1 | 优秀 |
| 2:1 | 良好 |
| 1.5:1 | 一般 |
| <1:1 | 避免 |

## 8. 决策检查清单

### 8.1 入场前检查
- [ ] 多时间框架趋势一致？
- [ ] 多个指标共振？
- [ ] 风险收益比 ≥ 2:1？
- [ ] 止损位明确？
- [ ] 仓位不超过推荐值？

### 8.2 持仓检查
- [ ] 趋势是否延续？
- [ ] 是否触及止损/止盈？
- [ ] 是否有更好的机会？
- [ ] 风险收益比是否恶化？

### 8.3 出场规则
1. 触及止损 → 无条件出局
2. 触及止盈 → 分批或全出
3. 趋势逆转 → 移动止损
4. 时间到期 → 重新评估

```bash
python run_framework_analysis.py
```

```python

framework = TechnicalInvestmentFramework()

# 获取建议
```
